📄 Day 1-A: Introduction to AI Automation & Agents

📝 Task 1: AI Automation কী?
AI Automation = AI দিয়ে repetitive কাজ automatically করানো
মানুষের হস্তক্ষেপ ছাড়া system নিজে সিদ্ধান্ত নেয় এবং কাজ করে

Real-life Example:
--> Email আসলে AI পড়ে → Category করে → Auto-reply পাঠায়
--> Network link down হলে AI detect করে → Alert পাঠায় → Ticket খোলে

📝 Task 2: AI Agent কী?
Agent = একটা AI system যে নিজে plan করে, সিদ্ধান্ত নেয়, কাজ করে
Tool ব্যবহার করতে পারে (API, web search, database)

3 ধরনের Agent:
Type            কী করে                            Example
Reactive        শুধু input এ react করে             Chatbot
Deliberative    Plan করে তারপর কাজ করে          Research Agent
Hybrid          দুটোই করে                         Network Monitor Agent
